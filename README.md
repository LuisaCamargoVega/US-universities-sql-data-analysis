# US-universities-sql-data-analysis
Exploratory Data Analysis (EDA) of Top U.S. Universities dataset using SQL.
Project Overview

This project performs an exploratory data analysis of the Top 50 U.S. Universities dataset using SQL queries.

The objective is to explore different characteristics of the universities, including:

Distribution of public and private universities

Employment rate of graduates

Research impact

Geographic distribution by state

Universities combining high research impact and employability

The analysis was conducted using SQL to extract insights from the dataset.
